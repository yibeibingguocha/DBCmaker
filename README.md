# DBC转化工具 v1.0.0.0

> 将标准CAN信号矩阵Excel文件转换为DBC格式的专业工具

## 📖 项目简介

本工具可以将汽车电子领域的CAN信号矩阵（Excel格式）自动转换为标准的DBC文件，支持：
- ✅ 多Sheet（多CAN路）处理
- ✅ 智能列名识别
- ✅ **收发方向区分**（新增）⭐
- ✅ 信号注释支持
- ✅ Motorola LSB/MSB和Intel字节序
- ✅ GBK编码（兼容CANdb++）

---

### v1.0.0.0+ (2026-05-21)

**新增功能：收发方向识别**

解决了不同CAN路之间同一报文收发方向不同的问题：

现在可以通过配置"收发方向"列，正确区分报文的发送和接收属性。

## 🚀 快速开始

### 1. 环境要求

- Python 3.6+
- openpyxl库

安装依赖：
```bash
pip install openpyxl
```

### 2. 运行工具

```bash
python DBC转化工具v1.0.0.0.py
```

或直接双击运行（如果已关联Python）

### 3. 使用步骤

1. **选择Excel文件**：点击"选择XLSX文件输入目录"
2. **确认列映射**：工具自动识别列名，可手动调整
3. **配置收发方向列**（可选）：指定T/R方向列
4. **选择字节序**：LSB / MSB / INTEL
5. **生成DBC**：点击"生成DBC文件"按钮

## 🎯 核心功能

### 1. 多Sheet支持

每个Sheet代表一个CAN网络或节点：
- Sheet名作为节点名
- 独立处理每个Sheet的报文
- 自动生成BU_节点列表

### 2. 智能列名匹配

自动识别多种列名格式：
- 中文：报文名称、报文ID、信号名称...
- 英文：Message Name、Message ID、Signal Name...
- 缩写：Msg Name、Can ID、Sig Name...

### 3. 收发方向识别 ⭐

**问题背景：**
同一CAN ID在不同网络中可能有不同的收发方向。

**解决方案：**
添加"收发方向"列，标记T（发送）或R（接收）。

**输出效果：**
```dbc
BO_ 256 EngineData: 8 ADASCAN
 SG_ RPM : 0|16@1+ (1,0) [0|0] "" Vector__XXX

BO_ 512 BrakeStatus: 8 BCAN
 SG_ Pressure : 0|8@1+ (1,0) [0|0] "" Vector__XXX
CM_ BO_ 512 "接收报文(R)";
```

### 4. 字节序支持

- **LSB**：Motorola最低有效字节（默认）
- **MSB**：Motorola最高有效字节
- **INTEL**：Intel字节序

### 5. 信号注释

支持为每个信号添加注释，在DBC中使用CM_属性保存。

## 🔧 高级配置

### 列名自定义

如果自动识别不准确，可以手动输入：
- 列字母：`A`, `B`, `C`...
- 列数字：`1`, `2`, `3`...
- 列名：`报文名称`, `Message Name`...

### TBOX节点支持

如果某列表头包含"TBOX"且值为"TX"，该报文归属于TBOX节点。

### 空行检测

连续5个空行会自动停止处理当前Sheet，避免无效数据。

## ⚠️ 注意事项

### DBC格式限制

1. **BO_只能指定发送节点**
   - 接收关系通过注释标记
   - 完整的接收节点列表需要在CANdb++中手动配置

2. **重复ID处理**
   - 同一ID在同一Sheet中会合并信号
   - 不同Sheet中的相同ID视为不同报文

3. **编码格式**
   - 使用GBK编码（支持21003个汉字）
   - 兼容CANdb++、CANoe等专业工具

### 最佳实践

1. ✅ 使用统一的列名规范
2. ✅ 报文ID使用十六进制格式（0x580）
3. ✅ 信号名称避免中文和特殊字符
4. ✅ 定期检查生成的DBC文件
5. ✅ 备份原始Excel文件

## 🐛 常见问题

### Q1: 为什么有些报文没有转换？

**A:** 检查以下几点：
- 报文ID是否为有效的十六进制
- 起始位和长度是否为数字
- 是否有连续5个空行提前终止

### Q2: 中文注释乱码怎么办？

**A:** 工具使用GBK编码，确保：
- Excel文件保存为UTF-8或GBK
- 用支持GBK的编辑器查看DBC文件

### Q3: 如何区分不同CAN路的同名报文？

**A:** 使用不同的Sheet，Sheet名作为节点名前缀。

### Q4: 收发方向列可以不设置吗？

**A:** 可以，不设置时所有报文默认为发送(T)，保持原有行为。
